Instructions

This application supports up to 28 days of exercise partitioned in 4 weeks. Description of the supported gestures/exercises in the next section below.
Fill only the fields of the days that you want to be included in your exercise routine and click the "Skip This Day" box of the other days.
All fields require non-negative integers (0, 1, 2, 3, ...). The Time fields refer to the duration (in seconds) of each repetition of the gesture/exercise.
The Sets Rest Time and Reps Rest Time fields refer to the duration (in seconds) of rest between consecutive sets and repetitions accordingly.
A day that is not to be skipped requires all its fields to be filled. If you do not want a field included (e.g., rest time), fill it with zero (0).
